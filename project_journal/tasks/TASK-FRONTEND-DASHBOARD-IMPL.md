# Task: Implement Frontend Dashboard

**Goal:** Implement the frontend dashboard for the SpreadPilot project as a responsive React application using Vite, TypeScript, and Tailwind CSS. The dashboard will provide an administrative interface, consuming data from the existing `admin-api` service via REST and WebSockets.

**Key Features:**
- Authentication (Login)
- Follower Management View
- Log Console
- Manual Commands

**Tech Stack:**
- React 18
- Vite
- TypeScript
- Tailwind CSS

**Backend API:** `admin-api` service
